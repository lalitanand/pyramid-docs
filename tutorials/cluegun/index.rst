======================================
 Shared Clipboard with repoze.cluegun
======================================

repoze.cluegun is a "pastebin" application, showing how you can use
the ``repoze.bfg`` framework for posting and managing paste sessions.

In this tutorial, you'll see:

  - Persistence

  - Handling form data

  - z3c.pt templating

  - Security
